# ZoomF

喜欢的小伙伴点个star多多支持

使用之前需要在配置文件添加zoomeye的api-key

执行： ./zoomF 生成config.ini配置文件。将api-key复制到''中即可

使用方法：
-w 查询语句
-q 查询页数
./zoomF -w app:"apache" -q 10
